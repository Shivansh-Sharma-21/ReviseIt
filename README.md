# ReviseIt: The JEE Diagnostic & Revision Tool

**ReviseIt** is an adaptive practice and diagnostic tool built specifically for JEE (Joint Entrance Examination) aspirants. Rather than serving as just another question bank, ReviseIt identifies "false confidence" in students—highlighting theoretical gaps they didn't know they had and offering optimized, level-appropriate questions that protect and improve their rank.

## 🎯 The Purpose
Many JEE aspirants struggle not because they lack study materials, but because they suffer from negative marking and false confidence. ReviseIt eliminates decision fatigue by providing a focused flow:
1. **Diagnosis (The Reality Check):** Exposing weak foundations before starting the main practice.
2. **Calibration:** Analyzing the gap between the student's self-assessed confidence and actual performance.
3. **Targeted Practice:** Dynamically serving questions categorized into Foundation (F), Core (C), and Stretch (S) based on diagnostic results, helping students close the gap and secure their rank.

## ✨ Features
- **Zero Decision Fatigue:** No scrolling through hundreds of modules or DPPs. Select the topic, and the flow adapts to you.
- **The Reality Check:** A mandatory checkpoint sequence to detect false confidence.
- **Confidence Calibration:** Adaptive difficulty adjustment depending on if your confidence aligns with your actual score.
- **Targeted Revision Pools:** A carefully curated mix of Foundation, Core, and Stretch questions mapped strictly to your session performance gaps.

## 🛠 Tech Stack
- **Frontend Framework:** React 19 + Vite
- **Styling:** Tailwind CSS V4
- **State Management:** React state & Context
- **Routing:** React Router DOM V7
- **Visualizations:** XYFlow Integration (React Flow)
- **Icons:** Lucide React

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd ReviseIt
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` to view it in the browser.

### Build for Production
To generate a production build:
```bash
npm run build
```
